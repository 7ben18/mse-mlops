# notebooks/

This directory is for exploratory and narrative notebooks only.

## Rules

- Organize notebooks by dataset or topic, for example `notebooks/ham10000/` or `notebooks/melanoma_cancer_dataset/`.
- Put reusable logic in `src/mse_mlops/`, especially `src/mse_mlops/analysis/`.
- Keep raw source data under `data/raw/`.
- Keep derived tables and intermediate analysis datasets under `data/processed/`.
- Keep generated reports and presentation artifacts under `reports/`.
